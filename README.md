# DR AI – Intelligent Deduction Resolution

> **Stop re-investigating invoices you already got right.**  
> DR automatically resolves routine deduction disputes and assembles complete case files for the deductions that actually require human review.

🌐 **Live Demo:** https://zen204.github.io/Deduction_Resolution/

---

## Overview

Accounts Receivable teams spend countless hours investigating deductions that were never actually complicated. Analysts manually search through invoices, purchase orders, contracts, shipping documents, and proof-of-delivery records to resolve disputes that often have an obvious explanation.

**DR** reimagines this workflow by using AI to classify deductions, surface supporting documentation, identify root causes, and automatically resolve cases that don't require human intervention.

Instead of forcing analysts to become detectives, DR gives them a completed investigation before they even open the case.

---

## The Problem

Most deduction management systems only record that a deduction occurred.

They don't answer questions like:

- Why did this deduction happen?
- Which department caused it?
- Does supporting documentation already exist?
- Can this deduction be resolved automatically?

As a result, AR teams spend hours rebuilding evidence for invoices that were already correct.

---

## The Solution

DR streamlines deduction resolution by:

- 🤖 Automatically classifying deduction types
- 📄 Matching invoices, contracts, purchase orders, and proof-of-delivery documents
- ⚡ Auto-resolving straightforward deductions
- 📊 Identifying recurring operational issues across Sales, Logistics, and Pricing
- 📁 Creating complete case files for disputes requiring human review

The goal isn't simply to process deductions faster—it's to eliminate unnecessary investigations altogether.

## Example Workflow

### Traditional Process

1. Receive deduction
2. Search ERP
3. Locate invoice
4. Find purchase order
5. Search email history
6. Verify shipping documents
7. Build evidence packet
8. Respond to customer

**Average Resolution Time:** ~9 days

### With DR

1. Deduction received
2. AI classifies dispute
3. Supporting documents automatically linked
4. Root cause identified
5. Simple deductions auto-resolved
6. Complex deductions delivered with a completed case file

**Target Resolution Time:** ~3 days

## Running Locally

Clone the repository:

```bash
git clone https://github.com/Zen204/Deduction_Resolution.git
cd Deduction_Resolution
```

Open `index.html` in your browser, or start a local development server:

```bash
python -m http.server
```

Then visit:

```
http://localhost:8000
```
